-------------------------------------------------------------
------------------ Projet de fin d'étude --------------------
------- Système de stabilisation motorisé pour caméra -------
-------------------------------------------------------------
       Réalisé par Sébastien Laporte à Polytech Tours
    Formation Informatique Industrielle par apprentissage
-------------------------------------------------------------

À la racine de ce dossier se trouvent l'ensemble des
livrables du projet.

  - Utilisateur : Un guide utilisateur est disponible dans
  ce dossier pour l'utilisation du système.

  - Administrateur : L'administrateur du système intervient
  pour ajuster les paramètres du système. Ce travail est
  nécessaire uniquement pour ajuster les paramètres en cas
  de modification de la structure mécanique, du type de
  moteur ou du type de caméra. Il a à sa disposition un
  guide administrateur et le code source du projet dont il
  aura besoin pour flasher à nouveau le programme dans le
  microcontrôleur avec les nouveaux paramètres

  - Développeur : Le développeur dispose de l'ensemble des
  codes sources mis à sa disposition dans le répertoire
  "Developpement" ainsi qu'un guide développeur.


-------------------------------------------------------------
==> Dossier "Developpement"
    Contient le programme du projet ainsi que l'ensemble des
    programmes réalisés au cours de ce projet.
=====> Dossier "Prototype initial"
       Contient les programmes Arduino réalisés au cours de
       la première partie du projet.
=====> Dossier "Prototype final"
       Contient les programmes Arduino réalisés au cours de
       la seconde partie de projet. Ils ont permis la
       réalisation et les tests pour le prototype final
=====> Dossier "Livrables"
       Contient les programmes Arduino mis en oeuvre pour
       la validation de chaque fonction tout au long du
       projet (Prototype initial et final). Ce dossier
       contient également la version finale du code et le
       répertoire pour la cross-compilation.
       Lien pour la mise en place de l'environnement de
       développement :https://www.pjrc.com/teensy/gcc.html
-------------------------------------------------------------
